ZoomingBox, a jQuery lightbox

While working for a client I had to develop this jQuery plugin to showcase products as it is done in ASOS

This is my first plugin, and it's first version therefore I encourage you to email me suggestions, issues and so on

So here it is, for all of you tu use, reuse, extend and contribute:

DEMO

complete image without thumbnails and both vertical and horizontal mouse action
$('.complete').zoomingBox();
Single image with thumbnails and both vertical and horizontal mouse action (Default)
$('.nothumb').zoomingBox({ thumbnails:false, });
vertical image with Thumbnails and vertical mouse action $('.vertical').zoomingBox({ zoom:'vertical', imageResize:'width', });
horizontal image with thumbnails and horizontal mouse action $('.horizontal').zoomingBox({ zoom:'horizontal' });
Gallery image 1 | Gallery image 2 | Gallery image 3 Gallery with vertical thumbnails $('.gallery').zoomingBox({zoom:'vertical',imageResize:'width',});
Gallery image 1 | Gallery image 2 | Gallery image 3 Gallery with horizontal thumbnails $('.gallery2').zoomingBox({zoom:'vertical',imageResize:'width',thumbnailPosition:'horizontal'});

Basic HTML Markup

Add necessary js and css files

Styles included in the css file are mostly basic functionality, customization of the plugin is possible through the API

<link href="css/zoomingBox.css" rel="stylesheet" type="text/css" />
<script src="http://code.jquery.com/jquery-latest.min.js"></script>
<script src="js/jquery.zoomingBox.js" type="text/javascript"></script>

Create anchor element

create an anchor element with the css class you want to target linking to the image you want to show in the lightbox

<a href="big image url" class="yourClass">your thumbnail or text</a>

If you want to use the gallery add the same "rel" attribute to every image you want to include in the gallery

<a href="big_image.jpg" class="yourClass" rel="myGallery">your thumbnail or text</a>
<a href="big_image2.jpg" class="yourClass" rel="myGallery">your thumbnail or text</a>
<a href="big_image3.jpg" class="yourClass" rel="myGallery">your thumbnail or text</a>

Fire up the plugin!

Simply target your anchor element's class using jQuery

$('.yourClass').zoomingBox();

API & options

Key Default value Description
zoom both This option defines what type of mouse movement will move the image in the lightbox. Available options are:both, vertical, horizontal
speed 800 Speed of the fade, time in milliseconds
thumbnails true Creates thumbnails within the zoomingBox
thumbnailPosition vertical Position for the thumbnails options are: vertical and horizontal
imageResize null available options are 'width', 'height' and null
containerWidth 70% Width for the lightbox, accepts percentage and px
containerHeight 90% Height for the lightbox, accepts percentage and px
closeButton true Appends a close button to the lightbox
closeText x Text to be displayed in the close button
nav true Boolean, displays navigation arrows
nextText 'NEXT >' Text to be displayed in the next button
prevText '< PREV' Text to be displayed in the previous button
overlayColor #000000 Overlay color accepts hex value
overlayOpacity 0.7 opacity for backdrop overlay
onStart null Will be called right before attempting to load the content
onComplete null Will be called after zoomingBox opens
onCleanup null Will be called right before closing
onClose null Will be called once zoomingBox is closed
